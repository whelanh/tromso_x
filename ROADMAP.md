# Tromsø Roadmap

**Last updated**: 2026-06-13 | **Status**: Alpha

Part of the [TunaOS](https://tunaos.org) ecosystem. BuildStream-based KDE Linux distribution.

## Alpha → Beta

- **Stabilize builds** — reproducible KDE desktop images from source
- **CI pipeline** — automated BuildStream builds on GitHub Actions
- **ISO publishing** — live ISO via tacklebox (tromso-iso now merged into tromso)
- **Documentation** — install guide, build instructions

## Planned

- **KDE Plasma 6.x** — track latest KDE releases
- **Flatpak integration** — Flathub preconfigured
- **Bootc migration** — evaluate moving from BuildStream to bootc-native image builds
